Welcome to "dynamic_content"

    Welcome your "dynamic_content"

    dynamic_content is a web framework and content management system implementation that aims to provide a very easy to use out-of-the box dynamic website as well as a highly configurable and flexible framework.

    This page is just a static webpage that has been created using the framework. It is a placeholder and eventually an instance of the framework itself will be running on this url.

    The project is still pretty young, but I'd encourage you to check it out on Github.

    I try to make this software to be the easiest to use and extend content management software there is.

    I hope you'll enjoy using this software. If you are a developer please consider helping out with the development, I am always looking for aid and fresh ideas.
